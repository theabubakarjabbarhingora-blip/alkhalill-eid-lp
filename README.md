# Al-Khalil Welfare - Ramadan Appeal

This project is a landing page for Al-Khalil Welfare's Ramadan Iftar Box appeal.

## About
Al-Khalil Welfare provides Iftar boxes to those in need during the holy month of Ramadan. Each box costs PKR 300 and provides a complete meal.

## Features
- Online Donation Form
- Responsive Design
- Ramadan Themed UI

## Development
1. Clone the repository.
2. `npm install`
3. `npm run dev`
